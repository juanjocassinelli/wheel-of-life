# 🌀 Wheel of Life

An interactive self-assessment tool to rate 8 key areas of life on a scale from 0 to 10.

**Features:**
- Live radar chart with stoplight colour coding
- Dark and light mode
- English and Spanish
- Download results as PDF

**Live site:** https://juanjocassinelli.github.io/wheel-of-life/
